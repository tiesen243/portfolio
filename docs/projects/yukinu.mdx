## Introduction

Yukinu is a modern, high-performance multi-vendor e-commerce platform designed for scalable online marketplaces. Its flagship frontend delivers seamless user and vendor experiences using the latest React and Next.js features, type safety, advanced SEO, and a flexible, extensible backend.

## Marketplace Features

- **Multi-Vendor Support**: Vendors can register, manage their own stores, and list products independently.
- **Product Catalog**: Advanced filtering, search, and category browsing across multiple vendors.
- **Shopping Cart**: Real-time, persistent cart supporting products from different vendors.
- **Vendor Dashboard**: Each vendor gets a dedicated dashboard for managing products, orders, and analytics.
- **User Authentication**: Secure registration, login, and sessions for both buyers and vendors, with support for social login and password recovery.
- **Checkout & Orders**: Smooth checkout flow, split orders by vendor, and order tracking.
- **Order Management**: Customers can view order history and track status; vendors can manage incoming orders and update their status.
- **Responsive Design**: Fully responsive, mobile-optimized design ensuring a smooth experience across all devices.
- **SEO & Marketing**: Landing pages, structured data, analytics, dynamic sitemap, Open Graph image generation, and robust metadata support.
- **Scalability**: Built with modern technologies and best practices to handle increasing traffic and data as the platform grows.

## Technical Highlights

- **Frontend — Web**: Next.js 16, React 19, server components, and App Router.
- **Frontend — Dashboard**: React Router 7, React 19, and Vite.
- **API Layer**: End-to-end typesafe communication with tRPC 11.
- **Backend & DB**: Modular backend (via `@yukinu/api`), Drizzle ORM, PostgreSQL, and scalable schema for multi-vendor data.
- **UI/UX**: Tailwind CSS v4, shadcn/ui, and Base UI for visually stunning, accessible components.
- **Email**: React Email components with Resend integration.
- **File Uploads**: UploadThing for file upload utilities.
- **Tooling**: Bun runtime, Turborepo for speedy builds, Oxc for linting and formatting, and a strict monorepo structure for maintainability.
- **Advanced SEO**: Dynamic sitemap, Open Graph image generation, and robust metadata support.
- **Automation**: Includes seed scripts and type-safe DB migrations via Drizzle Kit.

## Getting Started

### Prerequisites

- Node.js 20+ or Bun runtime (recommended: Bun 1.3.0+)
- Git
- Docker (for database setup)

### Setup Steps

```sh
git clone https://github.com/tiesen243/yukinu.git
cd yukinu
bun install

# Set up environment variables
cp apps/web/.env.example apps/web/.env
cp apps/dashboard/.env.example apps/dashboard/.env

# Set up the database
docker compose up -d db
bun --filter @yukinu/db run db:migrate

# Start all applications
bun run dev
```

- Web will be available at `http://localhost:3000`
- Dashboard will be available at `http://localhost:5173`
